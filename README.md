# Tyre Runner – Endless 3D Game

## Overview
A Unity-based endless runner game where the player controls a rolling tyre in a fast-paced environment.

## Uniqueness
The game is built around a **rolling tyre character**, making it different from traditional endless runner games.

## Key Features
- AI competitor racing against the player
- Swipe-based mobile controls
- Power-ups system (Magnet, High Jump)
- In-game character shop system
- Dynamic obstacle generation
- Progressive difficulty system

## AI Competitor
An intelligent AI runner competes with the player in real time to increase challenge and engagement.

## Power-ups
- Magnet: Attracts nearby rewards automatically  
- High Jump: Allows the player to jump over large obstacles  

## Controls
- Swipe Left/Right: Move between lanes  
- Swipe Up: Jump  
- Swipe Down: Slide  

## Technical Details
- Developed in Unity using C#
- Custom swipe input system
- AI lane-based movement logic
- PlayerPrefs used for saving progress and unlocks
- Modular and scalable code structure
